# accounting-app

Accounting remote for invoices, payments, and a ledger placeholder.

## Federation

Exposes:

```text
./Routes -> projects/accounting-app/src/app/remote-entry.routes.ts
```

The shell mounts this remote at `/accounting`.

## Commands

```bash
npm run start:accounting
npm run build:accounting
```

Standalone URL: `http://localhost:4203`
